# PRD: Система управління церквою УЦХВЄ

## Оригінальний запит
Створити сайт на основі бази даних церкви з файлу Microsoft Access (.mdb)

## Вибір користувача
- **Тип сайту**: Комбінований (публічна частина + адмінка)
- **Функції**: Всі - перегляд, пошук, редагування, статистика, дільниці
- **Авторизація**: З різними рівнями доступу
- **Тема**: Темна (темно-синя)

## Архітектура
- **Backend**: FastAPI + MongoDB
- **Frontend**: React + Tailwind CSS
- **Auth**: JWT з ролями (admin, presbyter, deacon, user)
- **База даних**: MongoDB (мігровано з Access)
