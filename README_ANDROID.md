# Як отримати Android додаток (APK)

Оскільки ви не програміст, ось найпростіший шлях перетворити цей код на файл `.apk` для телефону.

### Крок 1: Отримайте посилання на ваш сайт
Щоб створити додаток, ваш код повинен працювати в інтернеті.
*   Якщо ви використовуєте **CodePen**: Знайдіть кнопку "Change View" -> "Full Page View". Скопіюйте посилання з адресного рядка браузера.
*   **АБО** (Краще): Завантажте ці файли, заархівуйте їх та завантажте на будь-який безкоштовний хостинг (наприклад, Netlify Drop).

### Крок 2: Конвертуйте сайт у додаток (APK)
Використайте безкоштовний онлайн-сервіс, який "запакує" ваш сайт у файл Android.

1.  Відкрийте сайт **[WebIntoApp.com](https://www.webintoapp.com)** (або подібний, наприклад `appsgeyser.com`).
2.  Натисніть "Get Started" або "Create App".
3.  В поле **URL** вставте посилання з Кроку 1.
4.  Введіть назву: `Church Bus`.
5.  Натисніть **Next** -> **Make App**.
6.  Скачайте файл `.apk`.

### Крок 3: Встановлення
1.  Перекиньте файл `.apk` на ваш телефон (через Telegram, USB або Google Drive).
2.  Відкрийте файл на телефоні.
3.  Android запитає дозвіл на встановлення з невідомих джерел — надайте його.
4.  Натисніть "Встановити".

Тепер у вас є повноцінний додаток на телефоні!
